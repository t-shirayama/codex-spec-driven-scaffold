# Codex Skills

Codexに特定の作業手順を渡すための再利用可能なプロトコルを管理します。

## Available Skills

| Skill              | Purpose                                            |
| ------------------ | -------------------------------------------------- |
| `tech-debt-audit/` | 技術負債監査、コードヘルスチェック、保守性レビュー |

## Rule

- 汎用的に繰り返す作業だけをskill化する。
- skillにはトリガー、手順、出力形式、完了条件を書く。
