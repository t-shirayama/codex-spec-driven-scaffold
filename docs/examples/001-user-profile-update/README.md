# SPEC-001 User Profile Update

ユーザーが自分の表示名と自己紹介文を更新できる機能の完成済みexample specです。

## Files

| File                      | 役割                                   |
| ------------------------- | -------------------------------------- |
| `spec.md`                 | 背景、スコープ、振る舞い、ACと検証対応 |
| `plan.md`                 | 実装方針、リスク、検証方法             |
| `tasks.md`                | Codexが実行する作業単位                |
| `test-cases.md`           | ACに紐づくUT / IT / E2E / Manual       |
| `api.md`                  | API契約とエラー                        |
| `example-codex-prompt.md` | Codexへの依頼文例                      |

## Notes

- 実装可能な状態を示すため、`spec.md` は `status: approved` としています。
- 実プロジェクトでは、承認者、要件ID、APIパス、画面名をプロダクトに合わせて置き換えてください。
